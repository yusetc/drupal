Using background images

The easiest way to get a background image working in ckeditor is to:

  1. apply the appropriate background image class
  2. upload the image using ckeditor's image uploader
  3. switch to source code view
  4. copy the path to the image
  5. in the block level element you want the image applied to, add style="background-image: url(../url-to-image/image.jpg);"
  6. between the brackets (../) add the path you copied
  7. Delete the reference to the image that ckeditor created

Background positioning

by default, background images are centered horizontally and vertically and will expand (or shrink) to cover the entire background area.

You can position the start point of the background image by using
bk-img-tr (top right)
bk-img-tl (top left)
bk-img-bl (bottom left)
bk-img-br (bottom right)

Containing a background image

If you want the entire background image to be contained within the block level element, use one of the positioning classes with bk-img-contain.
